# Student Performance Analysis Dashboard

## Overview

This project analyzes student academic performance using Microsoft Excel. The goal is to identify the factors associated with students' performance in Mathematics, Reading, and Writing through data analysis, Pivot Tables, Pivot Charts, and an interactive dashboard.

## Objectives

- Clean and prepare the dataset for analysis.
- Analyze student performance across different categories.
- Identify patterns affecting academic scores.
- Build an interactive dashboard for better visualization.
- Generate meaningful business insights.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Visualization
- Dashboard Design

## Dataset Information

The dataset contains information about 1,000 students, including:

- Gender
- Race/Ethnicity
- Parental Level of Education
- Lunch Type
- Test Preparation Course
- Math Score
- Reading Score
- Writing Score

## Dashboard Highlights

The dashboard includes:

- Gender vs Scores
- Race/Ethnicity vs Scores
- Test Preparation vs Scores
- Lunch vs Scores
- Parental Education vs Scores
- Gender vs Lunch
- Interactive Slicers

## Key Insights

- Male students achieved slightly higher overall scores than female students.
- Students who completed the test preparation course performed better across all three subjects.
- Students whose parents hold a master's degree recorded the highest average scores.
- Students receiving a standard lunch scored higher on average than those receiving free/reduced lunch.
- Student performance varied across different race/ethnicity groups.
