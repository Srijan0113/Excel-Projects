# Hotel Booking Analysis Dashboard

## Overview

This project analyzes hotel booking data using Microsoft Excel to identify booking trends, customer behavior, cancellation patterns, and revenue-related insights. An interactive dashboard was created using Pivot Tables, Pivot Charts, and Slicers to help visualize key business metrics.

## Objectives

- Clean and prepare hotel booking data.
- Analyze booking patterns and customer behavior.
- Identify factors affecting cancellations.
- Build an interactive dashboard for decision-making.
- Generate actionable business insights.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Visualization
- Dashboard Design

## Dataset Information

The dataset contains over 119,000 hotel booking records with information such as:

- Hotel Type
- Booking Status
- Lead Time
- Arrival Date
- Customer Type
- Market Segment
- Country
- Average Daily Rate (ADR)
- Reservation Status

## Dashboard Features

The dashboard includes:

- Hotel Distribution
- Hotel Cancellation Analysis
- Monthly Booking Trends
- Market Segment Analysis
- Country-wise Bookings
- Customer Type Distribution
- Average Daily Rate (ADR) Analysis
- Cancellation vs Average Lead Time
- Interactive Slicers

## Key Business Insights

- City Hotels received significantly more bookings than Resort Hotels.
- City Hotels recorded a higher Average Daily Rate (ADR), indicating stronger revenue potential.
- Bookings with longer lead times were more likely to be cancelled.
- August recorded the highest number of bookings, while January had the lowest.
- Transient customers contributed the highest number of bookings.
- Online Travel Agencies (Online TA) generated the largest share of bookings.
- Portugal (PRT) was the leading source market with approximately 48.6K bookings.

## Business Recommendations

- Increase marketing efforts in Portugal, the highest-contributing country.
- Launch promotional campaigns during low-demand months to improve occupancy.
- Strengthen partnerships with Online Travel Agencies to maximize bookings.
- Monitor bookings with long lead times to reduce cancellation rates.
- Continue investing in City Hotels due to their higher demand and revenue potential.
